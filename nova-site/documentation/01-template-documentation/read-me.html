3CSS Structure

This template includes 8 css files. Let's take a look closely at our stylesheet files:

  • style.css - common css styles for this template.
  • bootstrap.css - standard bootstrap css styles.
  • pe-icon-7-stroke.css and helper.css - css styles for icons.
  • helper.css - modified from font-awesome helper CSS classes.
  • style-magnific-popup - contain css styles for the lightbox gallery.
  • animate.css- a cross-browser library of CSS animations
  • owl.carousel.min.css and owl.theme.default.min.css - css styles for the Owl carousel.

4JavaScripts

This template imports Javascript files. jQuery is a Javascript library that greatly reduces the amount of code that you must write. All the Javascript files can be found inside the js folder.

5Additional Info

Social Icons - please open icons-reference.html for the complete list of social icons.

Fonts Used - I have used 2 type of Google Font in this template: Cabin and Open Sans. If you would like to use different Google Fonts, you have to copy the font link in Google Fonts page and paste it in your index.html file.

Then you will need to add the new font name to your style.css file.
